Community

We welcome the involvement of the Elixir community, those familiar with GraphQL and those just starting to use it.

There are a number of ways you can get involved with the development of Absinthe and its related projects.

Report Issues, Make Suggestions, etc

Have you run into a problem with Absinthe? Have a bright idea on a change that could be made?

Please see our project listings and submit an issue to the appropriate project’s issue tracker – or contact us using one of the methods below.

Contribute Code & Documentation

If you’re using Absinthe for a project and have changes to submit, would like to improve documentation (either in the code projects themselves or this website), we’d welcome your assistance in making it even better.

You can find our projects in the GitHub absinthe-graphql organization. Contributing is just a pull request away.

Chat

We’d love to hear how you’re using Absinthe and help you dig deeper into what you can do with GraphQL and Elixir.

  • IRC - Find us on Freenode in #elixir-lang.
  • Slack - We’re in the Elixir #absinthe-graphql channel. For an invite to Elixir’s Slack team, go here.

Mailing List

We welcome questions and discussion via the absinthe-graphql mailing list.

Please report any issues & contribute changes to the website repository. Thanks!
