Using GraphiQL

The GraphiQL project is “an in-browser IDE for exploring GraphQL.”

We recommend using the pre-built Electron-based wrapper application, GraphiQL.app, which makes it very easy to introspect your development and production Absinthe applications.

GraphQL Hub

GraphQL Hub is an interesting website that you can use to introspect a number of public GraphQL servers, using GraphiQL in the browser and providing useful examples.

Please report any issues & contribute changes to the website repository. Thanks!
